## ⚡ Pro Tips (प्रो टिप्स)

### Windows में:
- **Alt + Arrow Keys**: विंडो को आकार देने के लिए
- **Ctrl + Shift + >**: फ़ॉन्ट आकार बढ़ाएं
- **Ctrl + Shift + <**: फ़ॉन्ट आकार घटाएं
- **Ctrl + Shift + ***: वर्तमान डेटा क्षेत्र का चयन करें
- **Ctrl + ~**: सूत्र मोड चालू/बंद करें
- **F4**: सेल संदर्भ को परिवर्तित करें ($ जोड़ें/हटाएं)

### Mac में:
- **Cmd + Option + Arrow Keys**: विंडो को आकार देने के लिए
- **Cmd + ]**: इंडेंट बढ़ाएं
- **Cmd + [**: इंडेंट घटाएं
- **Cmd + Shift + \**: वर्तमान डेटा क्षेत्र का चयन करें
- **Cmd + \`**: सूत्र मोड चालू/बंद करें
- **Cmd + Y**: फॉर्मूला दोहराएं

## 📝 महत्वपूर्ण नोट्स | Important Notes

- कुछ शॉर्टकट्स Excel के संस्करण के अनुसार भिन्न हो सकते हैं
- Mac पर **Cmd** आमतौर पर Windows पर **Ctrl** की जगह लेता है
- अपने कार्यप्रवाह को अनुकूलित करने के लिए शॉर्टकट्स को कस्टमाइज़ किया जा सकता है
- Excel विकल्प → कीबोर्ड शॉर्टकट्स में जाकर नई शॉर्टकट्स बनाई जा सकती हैं
- यह गाइड Microsoft Excel 365 और Excel 2021 के लिए सटीक है
